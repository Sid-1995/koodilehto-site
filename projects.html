Projects

Kryptoradio

Kryptoradio makes it possible to access the bitcoin network from anywhere, through DVB. It enables new ways to perform monetary transactions with minimal costs. Currently we are running trials on the technology at Finland.

Päätökset.fi - Jyväskylä

Päätökset.fi provides access to decisions made at Jyväskylä municipality in various forms. You can for instance check out the map to see which decisions affect you as a resident. Finnish only.

Personal Projects

JavaScript

jswiki

jswiki is perhaps the most comprehensive JavaScript package index at the web. It's one of those projects that grew out of a need. There are a lot of packages out there but they're all scattered. This wiki brings some order to the chaos.

caro.js

There are a lot of carousel plugins out there. Nothing quite suit this site perfectly. As a result caro.js was born. It's relatively light, customizable and scalable. It fits the fluid layout of this site really nicely.

ellipsis.js

This little plugin grew out of the GitHub commits widget visible at right. Sometimes the amount of commits per "push" could be quite big so it was quite messy visually. This plugin brought the situation under control.

yabox.js

yabox is another one of those little plugins that grew out of this site. It has been kept as light as possible by design. There are some hooks that make it possible to extend it on demand.

more.js

more.js is a CSS preprocessor that grew out of a blog post. It's more of a proof of concept at the moment.

bunit.js

bunit.js makes it simple to write unit tests using JavaScript. It provides the basic structure for testing based on RequireJS and a simple, chaining assert. It is possible to use some other means of asserting as well.

colorjs

colorjs provides a simple chaining API for dealing various color related operations (conversions and such).

qdemo

qdemo makes it easy to write interactive demo pages for jQuery plugins.

Python

Speccer

Speccer is one of those projects that makes unit testing a tiny bit easier. It provides minimal syntax and lets the programmer focus on the essential.

pyqa

pyqa makes it a lot easier to write those scripts that ask a bunch of things and then uses that information somehow.

pypandoc

pypandoc wraps pandoc, a universal document converter and makes it easy to use via Python.

iterplus

iterplus provides a set of functions that complement Python itertools.

Placidity

Placidity is an interactive interpreter that grew out of a series of blogposts. It sits on top of terminal and provides some additional functionality (variables and such).

PHP

TinyTest

TinyTest provides light syntax for writing PHP unit tests.

Unity 3D

Roboto

A 2d puzzle platformer game where an abandoned industrial robot named Roboto try to find his way out of the factory. Roboto benefits from his modular structure and he can change his torso and legs to adapt into the enviroment. The legs are useless when you have to cross high obstacles but how about the rocket engine!
